P8109

题目思路

既然要尽量多，也就是说，队伍通过少的题目对应气球较少的颜色，这样保证不会让通过多的只分到一点，而通过少的题目气球用不完。

题目代码

#include<bits/stdc++.h>
using namespace std;
int a[100005],b[100005];
int main()
{
    int n,ans=0;
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];//通过队伍数
    for(int i=1;i<=n;i++)cin>>b[i];//气球数
    //题目的输入已经排好序了。
    for(int i=1;i<=n;i++)
    {
        ans+=a[i];//每题通过每个队伍肯定要拿一只气球
        if(b[i]<a[i])ans-=a[i]-b[i];//如果不够，就少给一点气球
    }
    cout<<ans<<endl;
    return 0;
}
