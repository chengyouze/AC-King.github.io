1. 审题

不能被整除，说明 ，说简单点， 不是 的倍数。

2.思考

怎样才能让 不是 的倍数呢？

有一种很容易想到的办法： 只比 大一点，还达不到 的两倍。

那么就很好想到了， ，满足不能被整除以及递增数组的限制条件，而且题目中的 n 最大只有 1000 ，每次加一肯定不会超过 ，那么这题可以完结撒花了。

有个注意点，需要从 $ 开始累加，因为 2 是 1 的倍数。

void solve()
{
    int n;
    cin>>n;
    for(int i=2;i<=n+1;i++)//题解有说为什么是2到n+1

        cout<<i<<" ";
    }

}
