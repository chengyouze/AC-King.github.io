那么判断勾股定理代码为：
bool check(int a,int b,int c)
{
    //模拟冒泡排序，将abc排成从小到大排序。
    if(a>b)swap(a,b);
    if(b>c)swap(b,c);
    return a+b==c&&a>0&&b>0&&c>0;//判断，3条线段是否构成直角，并且长度大于0
}
